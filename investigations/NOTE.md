# About This Folder

This directory contains individual SOC alert investigation reports based on alerts analyzed in the LetsDefend platform.

Each file documents the investigation process for a specific SIEM alert, including:

• Alert overview and detection rule  
• Initial triage steps  
• Log and process analysis  
• Threat intelligence validation  
• Indicators of Compromise (IOCs)  
• MITRE ATT&CK technique mapping  
• Final incident classification (True Positive / False Positive)  
• Recommended response actions

The goal of these reports is to demonstrate practical Security Operations Center (SOC) skills, including alert triage, security log analysis, and incident investigation.

All investigations follow a structured analysis methodology similar to workflows used by SOC analysts when responding to security alerts.
